My journey in programming and interest in technology

My very first program was in JavaScript, and it was a bot for a game interface called Steam. I specifically used Node.js to create it. As I delved further into programming, my interest grew, and I started learning Python. In 2018, I created my first website using Python and a framework called Flask. One of the pages on my site allowed users to enter their height, which was then saved in a database using SQLite3. Once the user submitted their height along with their email address, they would receive an email showing the average height of all users in the database. Before starting my 10th year in the technical course of management and programming of computer systems, I also learned C++ to ensure that I could keep up with the course material. Currently, I have a solid foundation in programming and am comfortable working with Node.js, Laravel, JavaScript, C++, PHP, and MySQL for creating databases. I use Linux every day to program and complete my work, and frequently use Git to manage my programs. My journey in programming has been a continuous learning process, and I am always excited to explore new technologies and expand my knowledge in this ever-evolving field. I hope to continue growing and contributing to the world of technology through my skills and passion for programming.
