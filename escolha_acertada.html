006 : Escolha Acertada

Designed to obtain information using Python3. It scrapes data from the website and converts it into a structured CSV format for further analysis. Check my project here

The project was developed using Python 3 with Jupyter Notebook as the primary development environment. Beautiful Soup (bs4) was utilized to extract data, which was then processed using Pandas. Specifically, the data was transformed into a CSV format for ease of analysis and integration with other applications. By leveraging the capabilities of Jupyter, Beautiful Soup, and Pandas, the project provides a streamlined approach to data extraction and analysis. The resulting CSV file is a valuable resource for other data-driven applications and research projects.
